# WatsonDisasterApp


## Setting

create `environment.yaml` on your repository root.

```
kintone:
    domain: your_kintone_domain
    id: your_kintone_login_id
    password: your_kintone_password
watson:
    id: your_watson_id
    password: your_watson_password
twitter:
    token: your_token
    token_secret: your_secret
```
